# Dashboard App Frontend

Production-ready meeting management dashboard with AI-powered assistance.

## Features

- **Dashboard** with quick actions and "New Meeting" button
- **Settings** page with:
  - My Profile (name, role, company, background, voice sample)
  - My Teams (create teams with projects, goals, members)
  - Security settings
- **Meeting Creation** with team selection from Settings
- **Real-time AI Analysis** during meetings
- **Bilingual Support** (English/Hebrew with RTL)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts for analytics

## Deployment

Frontend deployed on Netlify: https://hemi-dashboard.netlify.app
Backend deployed on Railway
